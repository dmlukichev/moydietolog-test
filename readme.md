# Test task for "[Мой Диетолог](https://nodejs.org/)"

[Requirements](./readme.md)

Project is build with [create-react-app](https://github.com/facebook/create-react-app) without any specific customization.

Installation:
```
npm install
cp .env.dist .env
```

Start app:
```
npm run start
```
Run tests:
```
npm run test
```
